# Pokemon Team Builder

A simple, client-side web application to build and analyze Pokemon Showdown teams. Perfect for OU, Ubers, and other competitive formats.

## Features

- Build 6-Pokemon teams with type visualization
- Format selector for different competitive tiers (OU, UU, RU, NU, PU, Ubers, etc.)
- Team analysis showing weaknesses and resistances
- Export directly to Pokemon Showdown format
- 100% client-side - no server required
- Mobile-friendly responsive design

## Usage

Simply open the `index.html` file in your browser or visit the [live demo](https://riven-aigent.github.io/pokemon-team-builder/).

Click on empty slots to add Pokemon. The app includes sample data for popular OU Pokemon like Dragapult, Ogerpon, Iron Hands, Flutter Mane, Gholdengo, and Landorus-T.

## Privacy

This app is completely client-side. No data is ever sent to any server. Your team builds stay on your device only.
