# Pokemon Team Builder ⚔️

A Gen 9 OU team builder and analyzer built with SvelteKit 2 + Svelte 5.

## Features
- **Team Builder** - Select from 50 OU-viable Pokemon with full set customization
- **Ability, Nature, EVs, IVs** - Full competitive set builder for each slot
- **Move Selection** - Pick 4 moves with type indicators
- **Defensive Analysis** - See team-wide type weaknesses and resistances
- **Offensive Coverage** - Check which types your team can hit super-effectively
- **Threat Checklist** - Common OU threats (SD Ogerpon, sun teams, etc.) and whether you have answers
- **Core Detection** - Identifies known competitive cores in your team
- **Import/Export** - Showdown paste format support
- **Tera Type** - Select tera types for each Pokemon
- **Stat Calculator** - Live stat calculation with nature/EV visualization

## Tech
- SvelteKit 2 + Svelte 5 (runes)
- Static adapter for GitHub Pages
- Zero dependencies, no API calls
- Mobile-responsive
